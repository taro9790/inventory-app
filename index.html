スマホURL在庫管理

URLを入力して開ける在庫管理アプリです。公開用のURLから直接利用できます。

URLだけで使う場合

  1. この `index.html` を静的公開できる場所にアップロードしてください。
  2. 公開された URL をスマホブラウザで入力・アクセスします。
  3. HTTPS で開くと、カメラ読み取りが有効になります。

公開先例: GitHub Pages、Netlify、Vercel、Firebase Hostingなど。
